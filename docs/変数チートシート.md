# 変数チートシート

このシミュレータで頻出する主要変数の早見表です（現行実装ベース）。

## 1) 状態変数（時間で変わる）

| 変数 | 意味 | 主な場所 |
|---|---|---|
| `N` | 各株の個体数ベクトル（長さ=最大株数） | `backend/core.py`, `backend/engine.py` |
| `S` | 養分（基質）量 | `backend/manager.py`, `backend/core.py` |
| `T` | 毒素濃度 | `backend/manager.py`, `backend/core.py` |
| `pH` | 環境 pH | `backend/manager.py`, `backend/core.py` |
| `active_mask` | その株スロットが生存中かどうか | `backend/engine.py` |

## 2) 株の形質（traits 配列の7要素）

`traits[i] = [mu_max, Ks, p, r, T_opt, pH_opt, Rad_res]`

| インデックス | 変数 | 意味 |
|---|---|---|
| 0 | `mu_max` | 最大増殖速度 |
| 1 | `Ks` | モノド定数（低いほど低栄養で増殖しやすい） |
| 2 | `p` | 毒素生産能力 |
| 3 | `r` | 毒素耐性 |
| 4 | `T_opt` | 最適温度 |
| 5 | `pH_opt` | 最適 pH |
| 6 | `Rad_res` | 放射線耐性 |

## 3) 環境パラメータ（env_params）

`env_params = [temp, rad, k_tox, k_rad, k_acid, Y, d_T, hgt_prob, D, S_in]`

※ 後方互換で旧8要素形式も利用可（その場合 `D=0`, `S_in=S` 扱い）。

| インデックス | 変数 | 意味 |
|---|---|---|
| 0 | `temp` | 環境温度 |
| 1 | `rad` | 放射線レベル |
| 2 | `k_tox` | 毒素ストレス係数 |
| 3 | `k_rad` | 放射線ストレス係数 |
| 4 | `k_acid` | 酸性化係数（`dpH/dt` に効く） |
| 5 | `Y` | 収率（大きいほど S 消費が遅い） |
| 6 | `d_T` | 毒素減衰率 |
| 7 | `hgt_prob` | カーネル内HGT取り込み確率 |
| 8 | `D` | 希釈率（連続流入/流出の強さ） |
| 9 | `S_in` | 流入基質濃度（chemostat 供給濃度） |

## 4) 数値積分・時間管理

| 変数 | 意味 | 主な場所 |
|---|---|---|
| `dt` | 1ステップの時間刻み | `backend/manager.py` |
| `batch_size` | 1バッチで進めるステップ数 | `backend/manager.py` |
| `total_steps` | 累計ステップ数 | `backend/engine.py` |
| `steps` | カーネル側で回すループ回数 | `backend/core.py` |

## 5) コスト・成長関連

| 変数 | 意味 | 主な場所 |
|---|---|---|
| `g` | 成長率（モノド×温度ペナルティ×pHペナルティ） | `backend/core.py` |
| `m_costs` | 各株の維持コスト配列 | `backend/engine.py`, `backend/core.py` |
| `dN, dS, dT, dpH` | 微分方程式の右辺（時間変化率） | `backend/core.py` |

## 6) 分裂・変異・HGT

| 変数 | 意味 | 主な場所 |
|---|---|---|
| `threshold` | 分裂しきい値（`N >= threshold` で分裂） | `backend/engine.py` |
| `k_hgt` | 株間HGT率定数（manager側イベント） | `backend/manager.py` |
| `hgt_count` | 累計HGTイベント数 | `backend/manager.py` |
| `division_count` | 累計分裂イベント数 | `backend/manager.py` |
| `plasmid_pool` | 環境プラスミドプール（`[p, r, Rad_res]`） | `backend/engine.py`, `backend/core.py` |
| `pool_conc` | プール各スロット濃度 | `backend/engine.py`, `backend/core.py` |

## 7) 自動養分供給（最近追加）

| 変数 | 意味 | 主な場所 |
|---|---|---|
| `auto_feed_enabled` | 自動補給 ON/OFF | `backend/manager.py` |
| `feed_per_batch` | 1バッチごとの補給量 | `backend/manager.py` |
| `feed_max_s` | S の上限（補給クランプ） | `backend/manager.py` |
| `apply_nutrient_feed()` | 自動補給の実行関数 | `backend/manager.py` |

## 8) Adaptive dt（最近追加）

| 変数 | 意味 | 主な場所 |
|---|---|---|
| `adaptive_dt_enabled` | adaptive dt のON/OFF | `backend/manager.py` |
| `dt_min` | 最小時間刻み | `backend/manager.py` |
| `dt_max` | 最大時間刻み | `backend/manager.py` |
| `max_rel_change_per_step` | 1ステップ当たりの最大相対変化率（N基準） | `backend/manager.py` |
| `max_abs_s_change_per_step` | 1ステップ当たりの S の最大変化量 | `backend/manager.py` |
| `choose_adaptive_dt()` | 現在状態から dt を自動決定 | `backend/manager.py` |

## 9) WebSocket メッセージ型

### 受信（フロント → バックエンド）
- `RESUME` : シミュレーション再開
- `PAUSE` : 一時停止
- `SET_ENV` : `temp`, `rad` を更新
- `SET_FLOW` : `D`, `S_in` を更新（連続供給）
- `SET_ADAPTIVE_DT` : `enabled`, `dt_min`, `dt_max` を更新
- `SET_FEED` : `enabled`, `per_batch`, `max_s` を更新

### 送信（バックエンド → フロント）
- `BATCH_UPDATE` : `step`, `env`, `feed`, `ranking`, `pool`, `scatter`, `stats`

## 10) 安定デモ用プリセット（検証済み初期値）

50,000ステップ以上の安定動作を確認したデフォルト設定です。

### 初期株パラメータ（`engine.py` の `spawn(initial=True)`）
```python
traits = [0.4, 1.0, 0.0, 0.0, 25.0, 7.0, 0.0]
#         [mu_max, Ks, p, r, T_opt, pH_opt, Rad_res]
N0 = 500.0
```

| 変数 | 値 | 説明 |
|---|---|---|
| `mu_max` | 0.4 | 適度な成長速度 |
| `Ks` | 1.0 | 中程度の親和性 |
| `p` | 0.0 | 毒素生産なし |
| `r` | 0.0 | 毒素耐性なし |
| `T_opt` | 25.0 | 常温 |
| `pH_opt` | 7.0 | 中性 |
| `Rad_res` | 0.0 | 放射線耐性なし |
| `N0` | 500.0 | 初期個体数 |

### 環境パラメータ（`manager.py` の `__init__()`）
```python
S = 500.0
T = 0.0
pH = 7.0
env_params = [25.0, 0.0, 1.0, 1.0, 0.0, 100.0, 0.1, 0.005, 0.0, 0.0]
#             [temp, rad, k_tox, k_rad, k_acid, Y, d_T, hgt_prob, D, S_in]
auto_feed_enabled = True
feed_per_batch = 200.0
feed_max_s = 10000.0
```

| 変数 | 値 | 説明 |
|---|---|---|
| `S0` | 500.0 | 初期基質濃度 |
| `T0` | 0.0 | 初期毒素濃度 |
| `pH0` | 7.0 | 初期pH（中性） |
| `temp` | 25.0 | 常温環境 |
| `rad` | 0.0 | 放射線なし |
| `k_tox` | 1.0 | 毒素ストレス係数 |
| `k_rad` | 1.0 | 放射線ストレス係数 |
| `k_acid` | 0.0 | 酸性化なし |
| `Y` | 100.0 | 高収率（基質消費を抑制） |
| `d_T` | 0.1 | 毒素減衰率 |
| `hgt_prob` | 0.005 | HGT取り込み確率 |
| `D` | 0.0 | 連続供給なし |
| `S_in` | 0.0 | 流入基質濃度なし |
| `auto_feed_enabled` | True | **離散的自動供給を使用** |
| `feed_per_batch` | 200.0 | 各バッチで200の基質追加 |
| `feed_max_s` | 10000.0 | 基質上限 |

**注意**: このプリセットは実世界の生物学的値とは分離されており、シミュレーションの安定性と初心者向けの使いやすさを優先しています。
