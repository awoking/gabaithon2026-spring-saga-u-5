### 世代交代（上位生存＋変異）

新しい世代が誕生した時点で次世代とカウントする（決定的な一斉交代ではなく、緩やかな交代）。前の世代は時間経過で死滅しながら新しい世代へ移行する。

- **初期個体数：** 100体
- **GAMEOVER条件：** 全世代の個体数が0になる

### パラメータ

#### 🦠 細菌パラメータ（AI生成後は固定。世代交代時に変化する）

| パラメータ | 説明 |
|---|---|
| **寿命** | 時間経過で減少し、0になると死滅 |
| **最適環境** | 現在の環境との一致度をcos類似度で計算。高いほど突然変異率以外のパラメータが上昇し、低いと寿命の減少速度が増加 |
| **増殖速度** | 1秒あたりの個体増加数（計算式：個体増加数/s） |
| **耐環境ストレス** | 環境変化による寿命減少の絶対値を小さくする |
| **代謝効率** | 少ない栄養でも多くのエネルギーを得られる能力 |
| **突然変異率** | 100個体中何体が突然変異するかの確率（例：0.5 → 50体） |

**最適環境の詳細**

細菌ごとに最適な環境値と日光の好み（lover/hater）を持つ。

```json
"optimalEnv": {
  "temperature": 37,
  "nutrition": 0.6,
  "pH": 7.0,
  "sunlight": "lover"
}
```

現在の環境ベクトルと最適環境ベクトルのcos類似度でボーナス/ペナルティを決定。各環境パラメータは0〜1に正規化してからベクトル化する。

```
cos類似度 = 1.0        → 全パラメータ（突然変異率以外）+30%
cos類似度 = 0.8〜1.0   → 全パラメータ +10〜30%
cos類似度 = 0.5〜0.8   → 効果なし（ノーマル）
cos類似度 = 0.2〜0.5   → 寿命減少速度 +50%
cos類似度 = 0.0〜0.2   → 寿命減少速度 +200%
```

日光の好み（lover/hater）はcos類似度のボーナス/ペナルティに追加で乗算する。

---

#### 🌍 環境パラメータ（ステートレス、秒〜分ごとに常に変化する）

| パラメータ | 説明 | プレイヤー操作 |
|---|---|---|
| **温度** | 高いと増殖速度↑・寿命↓ | ✅ 雨を降らせて温度を下げる |
| **栄養量** | 少なくなると死滅が増加。代謝効率と連動 | ✅ 有機物を投与して栄養量を上げる |
| **pH** | 最適pHから外れると耐環境ストレスに影響 | ❌ 自動変化のみ |
| **放射線量** | 多いと突然変異率が上がる | ❌ 自動変化のみ |
| **日光量** | 時間帯で自動変化。多いと温度↑・栄養量↑（一時的） | ❌ 自動変化のみ |

**日光量の連鎖効果**

```
日光量↑ → 温度↑・栄養量↑（一時的）
日光量↓ → 温度↓・栄養量↓（一時的）
```

### グラフ

矢印ボタン [←][→] で切り替え。初期表示は個体数グラフ。

| グラフ | 記録単位 | 内容 |
|---|---|---|
| 個体数推移 | 日単位（1日=10分） | 全世代合計の個体数変化 |
| 温度 | 分単位 | 環境温度の変化 |
| 栄養量 | 分単位 | 栄養量の変化 |
| pH | 分単位 | pHの変化 |
| 放射線量 | 分単位 | 放射線量の変化 |

---

### 時間サイクル

1日 = 10分（リアル時間）で構成され、朝と夜に分割される。

| フェーズ | リアル時間 | 内容 |
|---|---|---|
| 🌅 朝 | 前半5分 | 日光量↑ → 温度↑・栄養量↑ |
| 🌙 夜 | 後半5分 | 日光量↓ → 温度↓・栄養量↓ |

---

### 世代交代ログ（ターミナル形式）

時間経過・世代交代・環境変化などがターミナル形式で流れる。

```
> 🌅 朝になりました。（Day 1）
> 🌙 夜になりました。（Day 1）
> 📅 1日が経過しました。現在の個体数：243体
> 第二世代となる細菌「わんわん菌」が誕生しました。
> ステータス：増殖速度が24→36になりました
```
