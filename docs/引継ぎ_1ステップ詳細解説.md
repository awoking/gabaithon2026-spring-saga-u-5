# 📍 1ステップ実行の流れ：完全な計算プロセス

## 概要

dt=0.01の**1ステップ**では、以下の処理が順次実行されます。

```
RK4 の4段階計算 → 状態変数の更新 → HGT判定
```

このドキュメントでは、最初の株が N=1000、栄養 S=200 の条件で、**実際に何が計算されるか**を具体例に沿って解説します。

---

### 初期条件

```
【株の情報】
Strain 0:

    │          μ_max  K_s   p    r   T_opt pH_opt Rad_res

  - T = 0.0        (毒素濃度)

```

## 2. RK4 4段階の詳細計算

### Stage 1: k1 の計算（t = 0.00 での微分）

#### A. 成長速度 g の計算

**数式：**
$$g_i(S, Temp, pH) = \mu_{max,i} \frac{S}{K_{s,i} + S} \times \exp\left( -\frac{(Temp - T_{opt,i})^2}{2\sigma_T^2} \right) \times \exp\left( -\frac{(pH - pH_{opt,i})^2}{2\sigma_{pH}^2} \right)$$

**代入値：**
- $\mu_{max,0} = 1.2$
- $K_{s,0} = 0.5$
- $S = 200.0$
- $Temp = 25.0°C$, $T_{opt,0} = 25.0°C$（$\sigma_T = 1.0$ と仮定）
- $pH = 7.0$, $pH_{opt,0} = 7.0$（$\sigma_{pH} = 1.0$ と仮定）

**Step 2a: モノド項**
$$\mu_{max} \frac{S}{K_s + S} = 1.2 \times \frac{200.0}{0.5 + 200.0} = 1.2 \times \frac{200.0}{200.5} = 1.2 \times 0.9975 = 1.1970$$

**Step 2b: 温度ペナルティ**
$$\exp\left( -\frac{(25.0 - 25.0)^2}{2 \times 1.0^2} \right) = \exp(0) = 1.0$$

→ 最適温度なので、ペナルティなし

**Step 2c: pH ペナルティ**
$$\exp\left( -\frac{(7.0 - 7.0)^2}{2 \times 1.0^2} \right) = \exp(0) = 1.0$$

→ 最適pHなので、ペナルティなし

**成長速度の結果：**
$$g_0 = 1.1970 \times 1.0 \times 1.0 = 1.1970 \text{ [1/h]}$$

---

#### B. ストレス死滅率の計算

**毒素死滅：**
$$d_{tox,0} = k_{tox} \max(0, T - r_0) = 1.0 \times \max(0, 0.0 - 0.0) = 0$$

**放射線死滅：**
$$d_{rad,0} = k_{rad} \max(0, Rad - Rad_{res,0}) = 1.0 \times \max(0, 0.0 - 0.0) = 0$$

→ 環境は清潔で、ストレスなし

---

#### C. 微分方程式の評価

**個体数の変化率：**
$$\frac{dN_0}{dt} = [g_0 - m_0 - d_{tox,0} - d_{rad,0}] \times N_0$$
$$= [1.1970 - 0.144 - 0 - 0] \times 1000$$
$$= 1.053 \times 1000 = 1053 \text{ [個体/h]}$$

**栄養の変化率：**
$$\frac{dS}{dt} = -\frac{1}{Y} \sum_i g_i N_i = -\frac{1}{10.0} \times 1.1970 \times 1000 = -119.7 \text{ [g/(L·h)]}$$

**毒素の変化率：**
$$\frac{dT}{dt} = \sum_i p_i N_i - d_T T = 0.0 \times 1000 - 0.1 \times 0.0 = 0$$

**pH の変化率：**
$$\frac{dpH}{dt} = -k_{acid} \sum_i g_i N_i = -0.05 \times 1.1970 \times 1000 = -59.85 \text{ [pH/h]}$$

**k1 の値：**
```
k1_N = 1053
k1_S = -119.7
k1_T = 0
k1_pH = -59.85
```

---

### Stage 2: k2 の計算（t = 0.00 + dt/2 = 0.005 での微分）

#### 中点での状態予測
$$N_{mid} = N_0 + 0.5 \times dt \times k1_N = 1000 + 0.5 \times 0.01 \times 1053$$
$$= 1000 + 5.265 = 1005.265$$

$$S_{mid} = S_0 + 0.5 \times dt \times k1_S = 200.0 + 0.5 \times 0.01 \times (-119.7)$$
$$= 200.0 - 0.5985 = 199.4015$$

$$T_{mid} = T_0 + 0.5 \times dt \times k1_T = 0 + 0 = 0$$

$$pH_{mid} = pH_0 + 0.5 \times dt \times k1_{pH} = 7.0 + 0.5 \times 0.01 \times (-59.85)$$
$$= 7.0 - 0.29925 = 6.70075$$

#### 中点での微分再計算

成長速度の再計算：
$$g_0^{(mid)} = 1.2 \times \frac{199.4015}{0.5 + 199.4015} \times \exp\left( -\frac{(25.0-25.0)^2}{2} \right) \times \exp\left( -\frac{(6.70075-7.0)^2}{2} \right)$$

$$= 1.2 \times 0.9975 \times 1.0 \times \exp\left( -\frac{(-0.29925)^2}{2} \right)$$

$$= 1.1970 \times \exp(-0.0448) = 1.1970 \times 0.9562 = 1.1443$$

**k2 の値：**（同様に計算）
```
k2_N ≈ 1001
k2_S ≈ -114.3
k2_T = 0
k2_pH ≈ -57.2
```

---

### Stage 3: k3 の計算（t = 0.00 + dt/2 = 0.005 での微分 with k2）

別の系列での中点計算：
$$N_{mid} = N_0 + 0.5 \times dt \times k2_N = 1000 + 0.5 \times 0.01 \times 1001 = 1005.005$$

（k2が k1 よりわずかに小さいため、k3 は k2 に近い値）

```
k3_N ≈ 1001
k3_S ≈ -114.3
k3_T = 0
k3_pH ≈ -57.2
```

---

### Stage 4: k4 の計算（t = 0.00 + dt = 0.01 での微分）

#### ステップ終了時点での状態予測
$$N_{final} = N_0 + dt \times k3_N = 1000 + 0.01 \times 1001 = 1010.01$$

$$S_{final} = S_0 + dt \times k3_S = 200.0 + 0.01 \times (-114.3) = 199.8557$$

$$T_{final} = 0$$

$$pH_{final} = 7.0 + 0.01 \times (-57.2) = 6.9428$$

#### 終了時点での微分

成長速度：
$$g_0^{(final)} = 1.2 \times \frac{199.8557}{0.5 + 199.8557} \times 1.0 \times \exp\left( -\frac{(6.9428-7.0)^2}{2} \right)$$

$$= 1.1973 \times \exp(-0.00163) \approx 1.1963$$

```
k4_N ≈ 1007
k4_S ≈ -119.6
k4_T = 0
k4_pH ≈ -59.8
```

## 3. RK4 重み付き和によるステップ更新

### 更新公式
$$y_{n+1} = y_n + \frac{dt}{6} (k_1 + 2k_2 + 2k_3 + k_4)$$

### 個体数の更新
$$N_0^{(new)} = N_0 + \frac{0.01}{6} (1053 + 2 \times 1001 + 2 \times 1001 + 1007)$$
$$= 1000 + \frac{0.01}{6} \times 6063$$
$$= 1000 + 10.105 = 1010.105$$

### 栄養の更新
$$S^{(new)} = S_0 + \frac{0.01}{6} (-119.7 + 2 \times (-114.3) + 2 \times (-114.3) + (-119.6))$$
$$= 200.0 + \frac{0.01}{6} \times (-717.8)$$
$$= 200.0 - 1.1963 = 198.8038$$

### pH の更新
$$pH^{(new)} = 7.0 + \frac{0.01}{6} (-59.85 + 2 \times (-57.2) + 2 \times (-57.2) + (-59.8))$$
$$= 7.0 + \frac{0.01}{6} \times (-357.25)$$
$$= 7.0 - 0.59542 = 6.40458$$

### ステップ終了時の値クリッピング
```python
# 負数を防ぐ
N = np.maximum(0.0, N)  # N_0 = 1010.105
S = np.maximum(0.0, S)  # S = 198.8038
```

## 4. HGT（水平伝播）処理

### HGT判定（確率的）

**条件：**
- `np.random.rand() < 0.01` → 1%の確率でHGT判定を実行

このステップでは判定されず、スキップされたと仮定します。

（判定された場合は、プラスミドプールからランダムに選択し、各株が確率的にプラスミドをコピーします）

## 7. 計算の重要なポイント

### ① RK4 の4段階が必要な理由

Euler法（RK1）では精度が不足：
```
RK1: N_{n+1} = N_n + dt × f(N_n)        誤差 O(dt²)
RK4: N_{n+1} = N_n + dt/6 × (k1 + 2k2 + 2k3 + k4)  誤差 O(dt⁵)
```

1万ステップ × 10,000バッチ = 1億ステップ実行する際、
RK1では数値誤差が蓄積して不安定になるため、RK4が必須。

### ② 環境フィードバックの階層性

**個体数 → 栄養消費 → 成長速度低下 → pH低下**：

- N が増加すると、$\sum g_i N_i$ が増加
- 栄養消費が増加 ($dS/dt = -\frac{1}{Y} \sum g_i N_i$)
- 栄養が枯渇すると g が低下（モノド項）
- 酸性化も同時に進行（$\frac{dpH}{dt} = -k_{acid} \sum g_i N_i$）

この**正の大きさのフィードバック**がシミュレーションの挙動を決めます。

### ③ 平衡点

十分に時間が経つと、成長と死滅が釣り合う平衡点に到達します：

$$g_i = m_i + d_{tox} + d_{rad}$$

上の例では $g_0 \approx 1.053$、$m_0 \approx 0.144$ なので、
成長が死滅を上回り、N は指数的に増加し続けます。

## 9. デバッグのコツ

### 値が NaN になったら
→ f_system() 内で割り算が起きている可能性
- Ks が 0 に近い：除数保護を確認
- 維持コスト計算：$1 / (Ks + 0.01)$ の "+0.01" が機能しているか確認

### 栄養が負になったら
→ `np.maximum(0.0, S)` でクリップされているが、
前ステップから栄養消費が多すぎる可能性
- Y（物質収率）が小さすぎないか確認
- 株数が爆増していないか確認

### pH が 0 未満になったら
→ 環境フィードバックが強すぎる可能性
- `k_acid` パラメータを減少させる
- 初期栄養 S を増やす
